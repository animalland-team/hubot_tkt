# hubot-tkt

A hubot script that does the things

See [`src/tkt.coffee`](src/tkt.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-tkt --save`

Then add **hubot-tkt** to your `external-scripts.json`:

```json
[
  "hubot-tkt"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
